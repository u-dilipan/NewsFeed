# NewsFeed

### Installation
1. Clone the repo / download
   ```sh
   https://github.com/u-dilipan/NewsFeed.git
   ```
2. Install NPM packages
   ```sh
   npm i 
   cd front-end
   npm i
   ```
3. Run the back end application
   ```sh
   npm start
   ```
4. Run the front end application
   ```sh
   cd front-end
   npm start
   ```
